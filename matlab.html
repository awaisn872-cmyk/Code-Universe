  • 2. Variables & Types
    • let name: string = "Alice";

    TypeScript variables have explicit types.

  • 3. Functions
    • function add(a: number, b: number): number {
    return a + b;
}
console.log(add(5,3));

    Functions can have typed parameters and return types.

  • 4. Classes & OOP
    • class Person {
    name: string;
    age: number;

    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log("Hello " + this.name);
    }
}

let p = new Person("Alice", 25);
p.greet();

    TypeScript supports classes, inheritance, and object-oriented programming.
