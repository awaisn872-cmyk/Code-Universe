  • 1. Introduction to Scala
    • object HelloWorld {
  def main(args: Array[String]): Unit = {
    println("Hello, Scala!")
  }
}

    Scala combines functional and object-oriented programming paradigms.

  • 2. Variables & Data Types
    • var name: String = "Alice" // mutable
val age: Int = 25           // immutable
val pi: Double = 3.14

    Use var for changeable values, val for constants.

  • 3. Functions
    • def add(a: Int, b: Int): Int = {
  a + b
}
println(add(5,3))

    Functions are reusable blocks of code.

  • 4. Classes & Objects
    • class Person(val name: String, var age: Int){
  def greet(): Unit = println(s"Hello $name")
}

val p = new Person("Alice", 25)
p.greet()

    Scala supports classes, objects, and OOP principles.

  • 5. Advanced Scala
    • Traits & Inheritance
    • Functional Programming (map, filter, reduce)
    • Pattern Matching
    • Collections & Generics
    val numbers = List(1,2,3,4,5)
val squares = numbers.map(x => x*x)
println(squares)
