  • 1. Introduction to Node.js
    • const http = require('http');

http.createServer((req, res) => {
  res.write('Hello, Node.js!');
  res.end();
}).listen(3000);

    Node.js allows building server-side applications using JavaScript.

  • 2. Modules & Packages
    • // Import a module
const fs = require('fs');

// Read a file
fs.readFile('file.txt', 'utf8', (err, data) => {
  if(err) throw err;
  console.log(data);
});

    Node.js uses modules to organize code and NPM to manage packages.

  • 3. Express.js
    • const express = require('express');
const app = express();

app.get('/', (req, res) => res.send('Hello, Express!'));

app.listen(3000, () => console.log('Server running on port 3000'));

    Express is a Node.js framework for building web apps and APIs.
